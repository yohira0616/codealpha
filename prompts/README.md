# prompts/

見積もり対話(claude -p)で使うプロンプトテンプレート。

| ファイル | 用途 |
|---|---|
| `task_breakdown.md` | 見積もり対話のベースプロンプト。要件テキストから機能・タスクを洗い出し、最後にタスク配列のJSONを出力させる。対話の初回にジョブが使用する。 |
| `estimate.md` | 既存タスク一覧の人日見直し用の追撃プロンプト。工程掛け率(基本設計・テスト・PM・予備費)の観点で抜けを指摘させる。PoCでは手動利用を想定。 |

## プレースホルダ

`ConversationReplyJob` が単純置換(単一パスの gsub)でプレースホルダを埋めてから claude -p に渡す。テンプレートエンジンは使わない。

| プレースホルダ | 置換内容 |
|---|---|
| `%{requirement}` | Project の要件テキスト(`requirement_text`) |
| `%{references}` | `references/` 直下の `*.md` をファイル名見出し付きで連結したもの(見積もり基準値などの参照データ)。ファイルが無ければ `(参照データなし)` |

`references/` に md を置くだけで次回以降の見積もり対話に反映される(サブディレクトリ・PDF は対象外)。
